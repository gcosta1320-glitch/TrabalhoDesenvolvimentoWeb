# Projeto Web

Este repositório contém páginas HTML e arquivos CSS do projeto.

Arquivos incluídos:
- index.html
- Contato.html
- Novidades.html
- Portfolio.html
- Sobre.html
- Stylecontato.css
- Stylenovidades.css
- Stylepaginaprincipal.css
- Styleportfolio.css
- Stylesobremim.css

Instruções rápidas:
1. Abra `index.html` no navegador.
2. Para publicar no GitHub, crie um repositório remoto e faça `git remote add origin <URL>` e `git push -u origin main`.
